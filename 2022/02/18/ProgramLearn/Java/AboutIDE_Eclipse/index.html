JAVA配置ECLIPSE(大坑)

坑

  • 入坑的原因在于lxf的学习插件用的是Eclipse
  • 所以我就下了一个，但是过程挺曲折的：
    1. 刚开始的时候下载的是官方的，镜像在韩国，没下下来
    2. 后来找个清华的镜像下载的

安装

  1. 安装的时候下载的是一个zip的绿色文件，翻镜像站的时候版本挺多的，选win32_x64那个版本就成，总之还是很快乐的。但是要500多M，长得一言难尽，像大型玩具
  2. 然后就包的导入，目录在Help--添加软件这个目录下面，怪，具体操作不难。
  3. 导入语言包的学习是通过：(128条消息) eclipse汉化教程（官方汉化包，傻瓜式操作，附带中英文快捷切换方式）_Zeromes的博客-CSDN博客_eclipse汉化————他同时还介绍了怎么切换中英文。
    • 另外比较好的：Eclipse安装中文语言包（官方汉化） - 知乎 (zhihu.com)
    • 放弃了，nmd根本分析不出来 垃圾ide
  4. 换源的操作见：(128条消息) eclipse 更换国内镜像_马帅的博客-CSDN博客_eclipse换源
  5. 适配高分辨率屏幕：(128条消息) eclipse适配高分辨率屏幕（不需要新建manifest）_seabiscuityj的博客-CSDN博客_eclipse高分辨率
  6. 黑色主题：(128条消息) eclipse设置 “暗黑色” 主题_zbctop的博客-CSDN博客_eclipse黑色主题

使用

  • 修改文件编码：eclipse编码格式设置 - AlanLee-Java - 博客园 (cnblogs.com)
